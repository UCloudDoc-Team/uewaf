# 白名单

白名单可以放行指定IP地址段内的地址的访问请求，不进行任何规则检测。  
白名单受黑白名单状态控制，仅当黑白名单状态为开启时生效。

![whitelist_img_1](/images/whitelist_img_1.png)

## 页面布局参数说明

  - IP内容：填充的具体IP值
  - 动作：拦截或者放行
  - 加入方式：自定义
  - 加入时间：加入该规则的时间
  - 备注：用户可以进行该字段的关键字搜索
  - 操作：编辑 删除
