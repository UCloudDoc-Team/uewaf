#  攻击详情
![](/images/15971370413247.jpg)

#### 参数说明
  - 最近攻击时间
  - 来源IP与目的IP
  - 请求路径
  - 地域
  - 攻击类型
  - 攻击次数
  - 工作模式
  - 匹配工作
  - 风险类型
  - 操作 
   * 详情 （点击查看，可以看到更多的请求信息）
   * 误报 （点击误报后，类似请求不再触发规则，即不再判断为攻击）
